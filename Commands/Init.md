# git init

Used to create empty repos.

```
mkdir NewDir
cd NewDir
git init
```

Once inside the repo, you can use commands such as:
[status](./Status.md)
[add](./Add.md)
[commit](./Commit.md)
[push](./Push.md)
and more.

Init creates the repo and encapsulates all content including other directories.
